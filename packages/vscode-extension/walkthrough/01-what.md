# Statewave — the project brain

Statewave makes your AI assistant aware of **this project**: structure, documentation, git history, code structure, conventions and decisions — retrieved on demand over MCP.

- **It never reads your assistant chat.** No interception.
- **Nothing is ingested until you explicitly run a command.** Preview-first, always.
- **Secrets are never indexed** (`.env`, keys, credentials) — a hard rule.
- **No telemetry. Vendor-neutral.** Works with Copilot, Cursor, Windsurf, Claude Code, Cline, Roo, Continue.
